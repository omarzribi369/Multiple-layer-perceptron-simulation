## 📖 About

**MLP Presentation** is an interactive educational tool that demystifies how neural networks learn. It visualizes every mathematical step of forward propagation, backpropagation, and gradient descent in real-time. Perfect for students, educators, and anyone curious about the inner workings of deep learning.

### 🎯 Why Use This?

- **See the Math in Action**: Watch numbers flow through the network and update in real-time
- **Step-by-Step Learning**: Control each iteration manually to understand the learning process
- **No Black Box**: Every weight update, gradient, and activation is explicitly shown
- **Interactive Exploration**: Change inputs, learning rates, and watch how the network adapts
